# VOD Fest 2026

## Task
- Erstelle eine Planung im Detail und Speichere die Informationen strukturiert im Verzeichnis ab. Lege hierzu eine sinnvolle Ordnungstruktur an
- Es soll eine Website für ein Music Festival erstellt werden
- Wenn möglich und sinnvoll in Wordpress
- Ich möchte das Du alles dabei übernimmst: Planung, Konzept, Design, Installation, Inhalte erstellen, Serverseitige installation
- Suche gute Beispiele für Websites für Musikfestivals
- Erstelle mir auch in einem Ordner direkt visuelle Ideen, wie die Seite aussehen kann

## Funktionen
- Ansprechende Home 
- Bilder und Infos zum Festival 2025
- Agenda für 2026
- Bandübersicht (alle Bands auf einer Seite)
- Für jede Band eine Einzelseite mit allen Infos zur jeweiligen Band
- Reise, Hotel und Öffentliche Transportmittel Information
- Feedbacks von Besuchern vom letzten Jahr
- Links zu allen Social Media Kanälen
- Kontaktinformationen
- Ticketkauf Informationen
- alles, was sonst auf einer top Musik Festival Seite notwendig ist
